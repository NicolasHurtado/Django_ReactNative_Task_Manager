## ğŸš€ Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd task_manager
   ```

2. **Install dependencies**:
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the application**:
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Run on device or emulator**:
   - Press `a` for Android
   - Press `i` for iOS
   - Scan the QR code with the Expo Go app (Android/iOS)
